STEPS
-----
1. Copy all files/folders above into your local groviatech repo,
   overwriting the existing ones with the same name.
2. In a terminal, inside the repo:
     git add .
     git commit -m "Wire up contact form/WhatsApp/social config, add SEO basics"
     git push
3. Netlify auto-deploys on push — check the Netlify dashboard for the
   build to go green (~30 sec for a static site).

ONE FILE TO EDIT LATER: js/config.js
-------------------------------------
When you have the real values, edit ONLY js/config.js:
  - whatsappNumber → your WhatsApp Business number (digits only, intl format)
  - formspreeId     → your Formspree form ID (sign up free at formspree.io)
  - social.instagram / linkedin / twitter → your profile URLs

Every page pulls from this one file now — no more hunting through 7 HTML
files to update a phone number.
